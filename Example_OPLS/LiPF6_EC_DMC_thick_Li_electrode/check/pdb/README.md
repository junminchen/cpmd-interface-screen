# PDB Check Folder

This folder centralizes key stage PDBs.

- `bulk/start.pdb`: bulk Packmol initial structure
- `bulk/final.pdb`: bulk NPT final structure
- `interface/electrolyte_start.pdb`: interface electrolyte-only structure
- `interface/start_with_electrodes.pdb`: assembled thick-Li interface structure
- `interface/start_with_electrodes_mc.pdb`: MC-equilibrated interface structure

Original paths are preserved as symlinks for workflow compatibility.
